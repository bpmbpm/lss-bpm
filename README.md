# Клонирование
Запускаем Git Bash. Выбор папки для клонирования репозитария, например, (обратный слеш)   
cd C:/Temp/logseq2   
git clone https://github.com/bpmbpm/lss-bpm   
Cloning into 'lss-bpm'...    
Проверяем наличие папок pages, logseq, .git   

Запускаем logseq, Меню: "Добавить новый граф".   
Выбираем папку (например, C:/Temp/logseq2).   
Можно использовать ручное обновление через расширение Git (A git plugin for logseq)
